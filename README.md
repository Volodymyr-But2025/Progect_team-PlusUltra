# 🛋️ Меблерія — Командний проєкт PlusUltra

Ласкаво просимо до репозиторію проєкту **«Меблерія»**! Це сучасний лендинг для
магазину меблів, створений у межах навчального курсу Full Stack Developer.

---

## 🚀 Про проєкт

Цей проєкт демонструє навички сучасної верстки, роботи з модульним JavaScript та
збирачем проєктів Vite. Основна мета — створити швидкий, адаптивний та зручний
інтерфейс для вибору меблів.

### 🛠️ Технологічний стек:

- **Інструменти:** Vite, Git, GitHub.
- **Верстка:** HTML5, CSS3 (BEM-методологія), Flexbox, CSS Grid.
- **Логіка:** JavaScript (ES6+), модульна структура.
- **Бібліотеки:** [Accordion-js](https://github.com/p-foucht/accordion-js),
  [Lenis](https://github.com/darkroomengineering/lenis) (плавний скрол).

## 🛠️ Як запустити проєкт локально

Щоб розгорнути проєкт на своєму комп'ютері, виконайте наступні кроки:

1. **Клонуйте репозиторій:**
   ```bash
   git clone [https://github.com/Volodymyr-But2025/Progect_team-PlusUltra.git](https://github.com/Volodymyr-But2025/Progect_team-PlusUltra.git)
   Перейдіть у папку проєкту:
   ```

Bash cd Progect_team-PlusUltra Встановіть залежності:

Bash npm install Запустіть режим розробки:

Bash npm run dev Після цього проєкт буде доступний за адресою
http://localhost:5173.

Збірка проєкту (Production):

Bash npm run build 📂 Структура проєкту Plaintext src/ ├── css/ # Стилі
(розділені за секціями) ├── img/ # Оптимізовані зображення (WebP, SVG) ├── js/ │
├── modules/ # Окремі JS-модулі (акордеон, модалки) │ └── main.js # Головний
файл ініціалізації └── index.html # Головна сторінка 📋 Основні розділи сайту
Hero: Головний банер з ідеальним пропозиціями.

Каталог: Зручна сітка товарів за категоріями.

Про нас: Історія та цінності бренду «Меблерія».

FAQ: Блок частих питань (реалізовано через акордеон).
